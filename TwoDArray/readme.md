
# 📂 2D-Arrays  (Java DSA)

Welcome to the **TwoDArray** module! This section is dedicated to mastering matrix manipulations, nested loop logic, and efficient data traversal in Java. 🚀

---

## 🚀 Overview
Working with 2D arrays is a fundamental skill in Data Structures. This folder contains various challenges—from basic sum calculations to advanced frequency counting logic.

### 🧠 Core Concepts Explored
* **Matrix Traversal:** Navigating rows and columns using nested loops.
* **Frequency Counting:** Tracking element occurrences using the **Visited Array** technique.
* **Optimization:** Analyzing time complexity ($O(N^4)$ vs $O(N^2)$) and space trade-offs.
* **Jagged Arrays:** Handling rows with different lengths.

---






## 📂 File Structure
| File Name | Description |
| :--- | :--- |
| `NumberOfOcc.java` | Finds frequency of every element using a visited matrix. |
| `SumAndAverage.java` | Logic for total sum and arithmetic mean of elements. |
| `ArraySortedCheck.java` | Efficient logic to check if elements follow a specific order. |
